# YouTube Trailer Automation

Automated YouTube trailer detection, processing, and re-upload pipeline.

## Features

- 🔍 **Auto-detect** new trailers from monitored YouTube studio channels
- ⬇️ **Download** trailers using yt-dlp
- 🎬 **Copyright protection** via FFmpeg (mirror flip, crop, speed, color adjustments)
- ✍️ **SEO-optimized content** generated by NVIDIA MiniMax AI
- 📤 **Auto-upload** to YouTube (5/day max, scheduled time windows)
- ☁️ **Google Drive backup** of all processed videos and metadata
- 📱 **Telegram notifications** for every step and daily summary
- ⏰ **GitHub Actions** cron job runs every 30 minutes

## Quick Start

### 1. Clone and Setup

```bash
git clone https://github.com/YOUR_USERNAME/youtube-trailer-automation.git
cd youtube-trailer-automation
chmod +x scripts/setup.sh
./scripts/setup.sh
```

### 2. Configure Credentials

Edit `.env` file with your credentials:

```bash
cp .env.example .env
nano .env
```

### 3. Google Cloud OAuth Setup

1. Go to [Google Cloud Console](https://console.cloud.google.com/)
2. Create a new project
3. Enable these APIs:
   - YouTube Data API v3
   - Google Drive API
4. Create OAuth 2.0 credentials
5. Download `client_secrets.json` and place in project root
6. Add these redirect URIs:
   - `http://localhost:8080`
   - `http://localhost:8080/`
   - `http://127.0.0.1:8080`
   - `http://127.0.0.1:8080/`

### 4. Generate OAuth Token

```bash
python scripts/generate_oauth_token.py
```

This will open a browser for Google OAuth consent. After authorization, the token is saved automatically.

### 5. Run

**Single run:**
```bash
python main.py --once
```

**Continuous (every 30 minutes):**
```bash
python main.py --interval 30
```

## GitHub Actions Setup

The automation can run automatically via GitHub Actions every 30 minutes.

### Required Repository Secrets

Add these secrets in your GitHub repository settings (Settings → Secrets and variables → Actions):

| Secret Name | Description |
|---|---|
| `GOOGLE_CLIENT_SECRETS` | Full JSON content of client_secrets.json |
| `GOOGLE_OAUTH_TOKEN` | Full JSON content of oauth_token.json |
| `YOUTUBE_CHANNEL_ID` | Your YouTube channel ID for uploads |
| `MONITORED_CHANNEL_IDS` | Comma-separated channel IDs to monitor |
| `GOOGLE_DRIVE_FOLDER_ID` | Google Drive folder ID for backups |
| `NVIDIA_API_KEY` | NVIDIA API key for MiniMax model |
| `TELEGRAM_BOT_TOKEN` | Telegram bot token |
| `TELEGRAM_CHAT_ID` | Your Telegram chat ID |
| `MAX_DAILY_UPLOADS` | Max uploads per day (default: 5) |
| `UPLOAD_TIME_WINDOWS` | Upload hours in IST (default: 9,12,15,18,21) |

## FFmpeg Copyright Protection

The following modifications are applied to each video:

| Modification | Default Value | Description |
|---|---|---|
| Mirror Flip | `true` | Horizontal flip |
| Speed | `1.05` | 5% faster playback |
| Crop | `3%` | Crop 3% from each side |
| Brightness | `0.02` | Slight brightness increase |
| Contrast | `1.02` | Slight contrast increase |
| Saturation | `1.03` | Slight saturation increase |

All values are configurable via `.env` file.
